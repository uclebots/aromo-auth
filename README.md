# aromo-auth
## Desarrollo
### Comandos
#### Una vez por creación de Vagrant VM
<pre>
    . setenv
    vagrant up
</pre>    

#### Destrucción Vagrant VM
<pre>
    vagrant destroy
</pre>    

#### Una vez por sesión de línea de comandos
<pre>
    . setenv
</pre>    
#### Iniciar/detener servicio
<pre>
    npm start
    npm stop
</pre>
#### Test
<pre>
    npm test
</pre>
