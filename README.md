# aromo-auth
## Desarrollo
### Comandos
#### Una vez por creación de Vagrant VM
<pre>
    . setenv
    vagrant up
</pre>    

#### Destrucción Vagrant VM
<pre>
    vagrant destroy
</pre>    

#### Iniciar servicio
<pre>
    vagrant ssh
    cd /vagrant
    npm start
</pre>
#### Detener servicio
<pre>
    vagrant ssh
    cd /vagrant
    npm stop
</pre>
#### Test
<pre>
    vagrant ssh
    cd /vagrant
    npm test
</pre>
